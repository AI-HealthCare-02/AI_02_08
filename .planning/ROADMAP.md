# 프로젝트 로드맵 (ROADMAP)

## 🏁 Phase 1: 기반 인프라 및 OCR/S3 연동 (1순위)
- **Phase 1.1**: AWS S3 버킷 설정 및 `aioboto3` 연동 모듈 개발.
- **Phase 1.2**: Naver Clova OCR API 연동 및 처방전 데이터 추출 로직(파서) 설계.
- **Phase 1.3**: 이미지 업로드 -> OCR 분석 -> 결과 반환 파이프라인 통합.

## 🏁 Phase 2: 약물 데이터베이스 구축 및 시딩 (2순위)
- **Phase 2.1**: `drugs.csv` 데이터 클렌징 및 매핑 스크립트 작성.
- **Phase 2.2**: Tortoise ORM을 사용한 대용량 데이터 시딩 기능 구현.
- **Phase 2.3**: `uv run` 명령어로 실행 가능한 시딩 자동화 스크립트 배포.

## 🏁 Phase 3: AI 기반 리포트 생성 및 OpenAI 연동 (3순위)
- **Phase 3.1**: OpenAI API 연동 및 복약 맞춤형 프롬프트 설계.
- **Phase 3.2**: OCR 데이터 + DB 정보를 조합한 리포트 생성 API 개발.
- **Phase 3.3**: 챗봇 지원을 위한 백엔드 인터페이스 확장.

## 🏁 Phase 4: 검증 및 고도화
- **Phase 4.1**: 전체 통합 테스트 및 성능 최적화.
- **Phase 4.2**: 예외 상황 처리(에러 로그, 재시도 로직) 강화.
- **Phase 4.3**: 최종 문서화 및 서비스 준비 완료.
