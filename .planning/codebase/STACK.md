# Tech Stack

## Backend
- **Language**: Python >= 3.13
- **Framework**: FastAPI
- **ORM**: Tortoise ORM
- **Database**: MySQL (using `aiomysql`, `asyncmy`)
- **Cache/Queue**: Redis
- **Security**: JWT, Passlib (bcrypt), Cryptography
- **Task Runner**: uv
- **AI/ML**: OpenAI, Scikit-learn, Sentence-transformers, PyTorch

## Frontend
- **Language**: TypeScript (using Vite)
- **Framework**: React 19
- **Routing**: React Router Dom
- **State Management**: React Hooks (native)
- **HTTP Client**: Axios
- **Linting/Formatting**: ESLint

## Tooling
- **Package Manager**: UV (Python), NPM/PNPM (JS)
- **Containerization**: Docker, Docker Compose
- **Linting**: Ruff (Python), ESLint (JS)
- **Type Checking**: MyPy (Python), TypeScript (JS)
- **Testing**: Pytest (Python)
